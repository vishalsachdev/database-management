# Cloud Databases

Databases hosted on cloud platforms.

## Benefits

- No infrastructure management
- Automatic scaling
- Built-in backup and recovery
- Global distribution
- Pay-as-you-go pricing

## Major Providers

- **AWS**: RDS, Aurora, DynamoDB
- **Google Cloud**: Cloud SQL, BigQuery
- **Azure**: Azure SQL, Cosmos DB
