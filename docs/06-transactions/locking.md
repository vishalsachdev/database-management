# Locking Mechanisms

Locks prevent concurrent transactions from interfering with each other.

## Lock Types

**Shared Lock (Read Lock)**: Multiple transactions can read
**Exclusive Lock (Write Lock)**: Only one transaction can write

## Locking Levels

- Row-level locking
- Table-level locking
- Page-level locking
