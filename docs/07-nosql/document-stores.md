# Document Stores

Document databases store data as JSON-like documents.

## Example: MongoDB

```javascript
{
  "_id": "12345",
  "name": "John Doe",
  "email": "john@example.com",
  "orders": [
    {"id": "001", "total": 50.00},
    {"id": "002", "total": 75.00}
  ]
}
```

## Use Cases

- Content management
- User profiles
- Product catalogs
