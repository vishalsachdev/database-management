# Introduction to NoSQL

NoSQL databases provide alternatives to relational databases for specific use cases.

## What is NoSQL?

**NoSQL** = "Not Only SQL" - databases that don't use traditional relational model.

## Why NoSQL?

- Horizontal scalability
- Flexible schemas
- High performance for specific access patterns
- Handle unstructured data
