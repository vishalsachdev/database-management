Diagram Specification: Product Development Timeline

Create an interactive timeline showing the major milestones in our product development from January 2023 to December 2025.

Events to include:
- January 15, 2023: Project kickoff and initial planning
- March 1, 2023: Design phase completed
- June 15, 2023: Prototype development started
- September 1, 2023: First prototype testing
- December 1, 2023: Beta version released
- March 15, 2024: User feedback integration
- June 1, 2024: Version 1.0 launched
- September 15, 2024: Feature expansion phase
- January 1, 2025: Version 2.0 development started
- June 30, 2025: Version 2.0 planned release

Requirements:
- Users should be able to zoom in to see detailed event information
- Include descriptions for each milestone
- Color-code by phase (Planning, Development, Testing, Launch)
- Make it responsive for mobile and desktop viewing
- Allow filtering by category

This will be used in a project management course to demonstrate product lifecycle.
