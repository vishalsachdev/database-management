## Overview

The workflow spans 8 steps across 4 categories:
- **Development:** Creating and updating skills
- **Installation:** Running install scripts and creating symlinks
- **Discovery:** Claude Code finding and loading skills
- **Usage:** Invoking and version controlling skills
